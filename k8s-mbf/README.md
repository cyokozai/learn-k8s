## 第一章 | 作ってみよう Kubernetes

### Docker

Docker はコンテナ仮想化技術の一つであり、コンテナエンジンのデファクトスタンダードといえるソフトウェアである。
Dockerfile, Docker image, Docker Hub とこれら一連の操作を提供する Docker CLI により、ユーザにとって一連のライフサイクルを扱いやすくしてくれている。
